Using the Spotify API and several python libraries, I created a machine learning model that attempted to “learn” the music taste of my friend, Antoine. You give this model any song, and it returns the probability that Antoine will like the song. A pretty accurate model, but could be improved. Next, I fed this model all of the songs that I like in my Spotify library. For each song it spat out a probability that Antoine would like it.

To tie it all together, I then built this app (my first one!) that generates playlists of songs from my library that Antoine would have a high probability of liking. You can select a mood and several genres to include in the playlist, and it will return up to ten songs. This is just a short demo, with a one-to-one recommendation of my own music to a friend. But if there was a way to accurately learn individuals’ music tastes on a large scale, this idea could be developed into a fun, new tool for peer-to-peer music recommendation.
